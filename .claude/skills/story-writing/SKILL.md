# Story Writing — 어린이 동화 시나리오 작성

storyteller 에이전트가 사용하는 시나리오 작성 스킬. 그림책 일러스트로 만들어질 것을 전제로 **장면 단위의 시각적 풍부함**을 최우선으로 한다.

## 모드 선택

| 모드 | 분량 | 구조 | 톤 |
|------|------|------|------|
| **picture-short** (기본) | 6~10 장면 | 단순 3막 | 잠자리·정서적 |
| **picture-long** | 25~36 페이지 | N챕터 × M페이지 | 흥미진진·모험·연속성 |

사용자 요청에서 분량 신호를 찾아 자동 결정:
- "30페이지 이상", "장편", "긴 동화", "챕터 동화", "모험 동화" → picture-long
- 그 외 또는 명시 없음 → picture-short

## Picture-Long 모드 (장편 동화)

장편은 호흡이 다르다. 한 장면 = 한 페이지였던 단편과 달리, 한 사건이 여러 페이지에 걸쳐 전개된다.

### 구조: 챕터 × 페이지

권장: **6 챕터 × 5 페이지 = 30 페이지** (또는 5×6=30, 5×7=35, 6×6=36)

각 챕터는 자체적인 작은 아크(도입-전개-마무리)를 가지며, 전체적으로는 큰 모험 아크를 형성한다.

### 6 챕터 아크 (어드벤처)

| 챕터 | 역할 | 페이지 수 | 분위기 |
|------|------|----------|--------|
| 1 | 일상 — 주인공들 소개, 평화로운 세계 | 5 | 따뜻한 |
| 2 | 사건의 발단 — 무언가 일어남 | 5 | 호기심·설렘 |
| 3 | 출발 — 모험에 나섬, 첫 시련 | 5 | 모험 |
| 4 | 깊은 위기 — 가장 큰 어려움, 새로운 동료/적 | 5 | 긴장 |
| 5 | 협동과 반전 — 함께 풀어냄, 작은 깨달음 | 5 | 환희·감동 |
| 6 | 귀환 — 변화된 일상 | 5 | 평온 |

(5×6=30 도 가능: 1=일상, 2=발단, 3=모험, 4=시련, 5=귀환)

### 페이지 단위 작성 규칙

- **한 페이지 = 한 이미지 + 짧은 본문** — 본문은 단편 모드와 같이 3~5문장
- 챕터 첫 페이지에는 챕터 제목과 약간 더 긴 도입부 (5~7문장 가능)
- 시각적 다양성 확보 — 같은 장소가 5페이지 연속이면 시점/각도/시간을 바꿈
- 다음 페이지로 넘어가고 싶게 만드는 미니 클리프행어 — 각 챕터 마지막 페이지에서

### 장편 동물 모험 가이드

- 동물 의인화: 옷·소품·도구를 갖되 동물의 본질적 특성(꼬리, 귀, 발톱)은 유지
- 3~5명의 주인공 일행 (각자 명확한 성격/역할)
- 위험 요소는 자연(폭풍, 강, 추위)이나 미스터리(사라진 것, 빛 등)로. 폭력 X
- 챕터마다 새로운 환경 — 숲, 강, 절벽, 동굴, 마을 등
- 동물 종마다 잘하는 것이 다른 점을 활용 (다람쥐 = 점프, 두더지 = 굴, 까치 = 비행)

### Picture-Long 출력 JSON 추가 필드

picture-short JSON 에 다음을 추가:

```json
{
  "mode": "picture-long",
  "chapters": [
    {
      "chapter_number": 1,
      "title": "챕터 제목",
      "summary": "한 줄 요약",
      "page_range": [1, 5]
    }
  ],
  "scenes": [
    {
      "scene_number": 1,
      "chapter_number": 1,
      "is_chapter_opener": true,
      ...
    }
  ]
}
```

## 동화 구조 (3막 8장면 권장) — picture-short 모드

| 장면 | 역할 | 톤 |
|------|------|------|
| 1 | 도입 — 주인공과 일상 세계 소개 | 따뜻한 |
| 2 | 부름 — 호기심 자극하는 사건 | 신비로운 |
| 3 | 출발 — 주인공이 모험 시작 | 설레는 |
| 4 | 첫 만남 — 새로운 친구/존재 등장 | 놀라운 |
| 5 | 시련 — 작은 어려움이나 갈등 | 긴장된 |
| 6 | 깨달음 — 도움/지혜를 얻음 | 감동적인 |
| 7 | 해결 — 문제를 풀어냄 | 환희의 |
| 8 | 귀환 — 따뜻한 마무리 | 평온한 |

(장면 수가 다르면 비례 조정. 6장면이면 4·5·6을 통합)

## 작성 원칙

1. **장면 본문은 짧게** — 3~5문장. 그림책은 텍스트보다 그림이 먼저.
2. **시각 단서 풍부하게** — 색채(노을빛, 푸르스름한 달빛), 움직임(폴짝, 살랑살랑), 표정(눈을 동그랗게, 입꼬리 올리며).
3. **반복과 리듬** — 어린이 친화. 의성어·의태어 자연스럽게.
4. **캐릭터 외모 고정** — 첫 등장 시 외모를 확정. 머리색·옷·소품 한두 가지를 모든 장면에서 유지.
5. **갈등은 작게** — 죽음, 폭력, 진짜 위협 금지. "길을 잃었다", "별이 떨어졌다", "친구가 슬퍼했다" 정도.
6. **메시지는 가볍게** — 교훈을 직설적으로 말하지 않고 상황으로 보여준다.

## 추천 주제 풀 (사용자 입력 없을 때 자율 선택용)

- 별을 모으는 작은 동물
- 바다 밑 친구를 찾아가는 모험
- 구름을 만드는 마을의 아이
- 잃어버린 색깔을 되찾는 여행
- 달빛 도서관의 책 정령
- 계절을 바꾸는 작은 마법사
- 그림자와 친구가 된 아이
- 노래하는 씨앗
